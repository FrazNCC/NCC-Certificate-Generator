# Certificate Generator

A professional certificate generation tool supporting both single and bulk certificate creation.

## Features
- Single certificate generation with real-time preview
- Bulk certificate generation from CSV files
- Professional canvas-based design with gold borders
- High-quality landscape PDF output
- ZIP download for bulk certificates
- 100% client-side processing

## Quick Start
1. Open `index.html` in your browser
2. Fill in certificate details
3. Preview and generate PDF

## Deployment to GitHub Pages
1. Create a GitHub repository named `certificate-generator`
2. Upload all files from this folder to the repository root
3. Go to Settings â†’ Pages
4. Set Source: main branch, root folder
5. Your app will be live at: `https://yourusername.github.io/certificate-generator/`

## CSV Format
For bulk certificates, prepare a CSV file with these columns:
- Student Name
- Course Name  
- Lecturer Name
- Issue Date (YYYY-MM-DD format)
- Certificate Type (Completion, Achievement, or Excellence)
